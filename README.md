## 📖 Sobre o Projeto

Portfólio profissional criado para apresentar minhas habilidades, projetos e formas de contato de maneira clara e visualmente atraente.

O principal destaque é a integração com a **Brasil API**, uma API REST desenvolvida com **Node.js** e **Express**, que fornece dados sobre estados, capitais, cidades e regiões do Brasil.

---

## ✨ Funcionalidades

- ✅ Apresentação do desenvolvedor  
- ✅ Exibição de habilidades técnicas  
- ✅ Listagem de projetos  
- ✅ Integração com Brasil API  
- ✅ Informações de contato  
- ✅ Navegação entre seções  
- ✅ Layout totalmente responsivo

## 🚀 Como Visualizar Localmente

Não é necessário instalar dependências. Basta clonar e abrir o arquivo HTML:

```bash
git clone https://github.com/pedsousa06-ai/Portifolio-Projetos.git
cd Portifolio-Projetos
```

Em seguida, abra o arquivo `src/index.html` diretamente no navegador.

## 📊 Projeto em destaque

### 🇧🇷 Brasil API

API REST que permite:

- 📍 Listar todos os estados  
- 🔎 Consultar dados de um estado  
- 🏙️ Buscar capitais  
- 🌎 Filtrar estados por região  
- 🏘️ Listar cidades de um estado  

🔗 Repositório: https://github.com/pedsousa06-ai/Api-Brazil
